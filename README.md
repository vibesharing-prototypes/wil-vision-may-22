# Custom attributes — full vision (v2)

Standalone Vite + React + Atlas app. This is the **full vision** prototype: primary schema management, roles, and workflows. Explorations (BOS, form preview, permission cards, read-only schema viewer) ship in **`custom-attributes-lab/`** only.

```bash
npm install
npm run dev
```

**Related apps:** [`../PROTOTYPES.md`](../PROTOTYPES.md) — MVP and lab sibling folders, deploy URLs, stakeholder wording. Optional env: `VITE_MVP_PROTOTYPE_URL` for the in-app banner link to the MVP deploy.
